- [gRPC](https://grpc.io/)

- [JWT](https://jwt.io/)
- [Express](https://expressjs.com/)
- [Mongoose](https://mongoosejs.com/)

## :wrench: Configuration

#### 1. Install all dependencies with

```sh
$ npm install 
```

or using yarn

```sh
$ yarn
```

#### 2. Start microservice

```sh
$ docker-compose up
```
